org.kde.discover.notifier — KDE Discover update notifier that checks for and shows available software updates.

org.kde.kdeconnect.daemon — Phone integration service for notifications, clipboard sync, and file transfer between devices.

plasma-kactivitymanagerd.service — Manager for KDE Activities, which provide separate task-based workspace environments.

plasma-kaccess.service — Accessibility helper that monitors and notifies about features like sticky keys.

baloo indexer — File indexing service that scans your files so they appear instantly in KDE search.

plasma-baloorunner.service — KRunner plugin that lets the launcher search through the Baloo file index.

plasma-ksystemstats.service — Background collector for CPU, memory, disk, and sensor metrics used by system monitor widgets.

obex.service — Bluetooth OBEX file transfer service used for sending/receiving files over Bluetooth.
